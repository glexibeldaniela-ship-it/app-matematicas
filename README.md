# App Matemáticas - Plataforma de Exámenes

## Descripción
Plataforma web para que estudiantes realicen exámenes y profesores gestionen evaluaciones.

## Tecnologías
- HTML5
- CSS3
- JavaScript Vanilla
- Firebase (Autenticación y Firestore)

## Estructura del Proyecto
```
app-matematicas/
├── index.html
├── registro.html
├── css/
│   ├── estilos-generales.css
│   ├── login.css
│   ├── dashboard-estudiante.css
│   └── dashboard-profesor.css
├── js/
│   ├── firebase-config.js
│   ├── autenticacion.js
│   ├── estudiante.js
│   ├── profesor.js
│   └── utilidades.js
├── estudiante/
│   ├── dashboard.html
│   ├── hacer-examen.html
│   └── mis-calificaciones.html
└── profesor/
    ├── dashboard.html
    ├── crear-examen.html
    ├── gestionar-estudiantes.html
    ├── calificaciones.html
    └── reportes.html
```
